# Akan Autocomplete / Text Prediction

A research-to-product system for next-word prediction and character-level
autocomplete in Asante Twi, a low-resource Ghanaian language spoken by ~44%
of Ghana's population.

## Quick Start

```bash
uv sync
python scripts/prepare_dataset.sh
python scripts/run_experiment.sh EXP-04
```
